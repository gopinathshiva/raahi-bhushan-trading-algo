    How to get the IDs

    1. Go to the Hall of Fame page on Sensibull
      (live positions, sorted by P&L desc)
    2. Wait for the table to fully load
    3. Open DevTools → Console tab (F12)
    4. Paste and run the script below
    5. IDs are auto-copied to clipboard — paste here
    // Run on:
// https://web.sensibull.com/verified-pnl/hall-of-fame
// ?live_page=1&live_sort_by=pl_desc&category=live_positions

const ids = [];
const rows = document.querySelectorAll(
  '#tabpanel-live_positions div div:first-child table tbody tr'
);
rows.forEach(row => {
  const cell = row.querySelector('td:nth-child(2)');
  if (cell) {
    const text = cell.innerText.trim();
    if (text) ids.push(text);
  }
});

console.log("Sensibull IDs found:", ids.length);
copy(JSON.stringify(ids)); // auto-copies to clipboard
